# iflow_tomato 项目

## 项目概述

**多任务队列计时闹钟** - 一个基于番茄工作法的 Windows 桌面计时工具

## 功能特性

### 核心功能
- 多任务队列管理（顺序执行）
- 每个任务支持：名称、预估时长、休息次数、休息时长
- 工作/休息周期自动切换
- Windows 系统通知栏 Toast 消息提醒

### 任务管理
- 添加/删除任务
- 批量导入任务（支持 `- [ ]` 前缀格式）
- 替换所有任务 / 新增到现有任务
- 清空所有任务

### 控制功能
- 开始/暂停/继续
- 跳过当前任务（静默执行）
- 修改任务（保留当前进度）

### 日志统计功能（v1.1 新增）
- **自动记录**：任务完成/跳过时自动保存到本地 JSON 文件
- **存储方式**：按日期分文件存储（`logs/YYYY-MM-DD.json`）
- **日历视图**：月历样式，支持月份切换
- **颜色标记**：
  - 🟩 深绿：全部完成
  - 🟨 黄色：有未完成任务
  - ⬜ 灰色：无任务
- **月度统计**：任务数、完成率、专注时长、休息时长等
- **日志导出**：支持导出为 CSV 格式

### 视觉设计
- 黑/白/灰主色调
- 绿色时间数字（#22C55E）
- 当前任务：深绿色高亮框
- 已完成/跳过任务：浅绿色背景
- 表头固定吸顶
- 自定义窗口图标（时钟样式）

```

├── TaskItem          # 任务数据结构
│   ├── name, duration, break_count, break_duration
│   ├── completed, skipped
│   └── to_dict()     # 序列化为日志记录
├── Logger            # 日志记录器
│   ├── log_task()           # 记录任务完成/跳过
│   ├── get_month_logs()     # 获取月度日志
│   ├── get_month_statistics() # 月度统计
│   └── export_month_to_csv() # CSV导出
└── TimerApp          # 主程序
    ├── _create_ui()         # 构建界面
    ├── _start_queue()       # 开始队列
    ├── _run_timer()         # 倒计时逻辑
    ├── _show_toast_notification() # Toast 通知
    ├── _show_calendar_window() # 日历统计窗口
    └── _on_closing()        # 关闭清理
```

## 使用说明

### 源码运行
```bash
python timer_app.py
```

### EXE 运行
双击 `dist\多任务队列计时闹钟.exe`

### 日志存储
- 日志自动保存在 EXE 所在目录的 `logs/` 文件夹
- 每天一个 JSON 文件，按日期命名
- 批量替换任务会清除当天日志

## 备注

- 用户使用 **Windows** 系统
- 项目名称 "tomato" 来源于番茄工作法（Pomodoro Technique）
- 无需 Python 环境即可运行打包后的 EXE
- 分发时只需发送 EXE 文件，日志会在使用时自动创建
- 代码无第三方依赖，仅使用 Python 标准库
